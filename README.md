# projeto-fazenda-fict-cia
Projeto Fazenda Fictícia – Análise Ambiental Aplicada ao Crédito Rural

Sobre o projeto

Este projeto apresenta uma análise ambiental geoespacial aplicada a uma área agrícola fictícia, com o objetivo de demonstrar como ferramentas de geoprocessamento e sensoriamento remoto podem auxiliar na avaliação ambiental para concessão de crédito rural.

A proposta do estudo é integrar diferentes camadas de informação ambiental para apoiar a tomada de decisão, considerando critérios como vigor da vegetação, disponibilidade hídrica, uso e cobertura do solo, relevo e histórico de distúrbios ambientais.



Objetivo

Simular uma análise ambiental aplicada ao crédito de custeio agrícola, avaliando se a área de estudo atende a critérios ambientais e climáticos básicos que podem influenciar a liberação de financiamento rural.



Dados e fontes utilizadas
	•	Sentinel-2 (Copernicus) – cálculo de NDVI
	•	MapBiomas – uso e cobertura do solo e cicatrizes de fogo
	•	INMET / SISAGRO – dados agrometeorológicos e balanço hídrico
	•	Copernicus DEM (30 m) – análise de declividade
	•	IBGE – malhas territoriais



Ferramentas e softwares
	•	QGIS
	•	Python (scripts para cálculo de NDVI)
	•	Blender (visualização 3D)
	•	Google Sheets (organização de dados meteorológicos)



Etapas da análise
	•	Delimitação da área de estudo e elaboração do mapa de localização
	•	Cálculo do NDVI para avaliação do vigor da vegetação
	•	Análise de balanço hídrico a partir de dados meteorológicos
	•	Classificação do uso e cobertura do solo
	•	Análise de declividade para verificação de restrições ambientais
	•	Identificação de cicatrizes de fogo recentes
	•	Visualização tridimensional do terreno e da vegetação



Resultados

Os resultados indicaram predominância de vegetação com bom vigor, ausência de déficits hídricos significativos no período analisado, uso do solo majoritariamente associado a pastagens e formações vegetais, relevo favorável à atividade agrícola e ausência de cicatrizes de fogo relevantes na área de estudo.



 Limitações
	•	Área de estudo fictícia
	•	Uso de estação meteorológica pontual
	•	Resolução espacial limitada de alguns produtos
	•	Ausência de dados agronômicos específicos da cultura
